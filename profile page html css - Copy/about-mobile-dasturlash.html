Mobile Dasturlash

Oʻyin-kulgi dasturlari (tillarni oʻrganish, kitoblarni oʻqish, sayohat qilish, sport), xarid qilish ilovalari (e-tijorat segmenti) va alohida toifadagi mobil oʻyinlar mavjud
